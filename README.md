# video-game-user-prediction
DSCI 100 Individual Project: Predicting Minecraft Server Usage

## Objective
- Explore the Minecraft server data collected by the UBC research team and led by Frank Wood
- Answer the broad question "What player characteristics and behaviours are most predictive of subscribing to a game-related newsletter, and how do these features differ between various player types?"
- Formulate a specific question that supports the broad question using explanatory variables in the dataset

## Content
1. Data Description
2. Questions
3. Exploratory Data Analysis (EDA) and Visualization
4. Methods and Plan
5. GitHub Repository Link
